# Pittsburgh History Interactive Website

Explore Pittsburgh’s rich history through an interactive, user‑friendly web experience. This project was built using React, HTML, CSS, and JavaScript, and features an intro page, an interactive neighborhood map, and a fun facts section highlighting unique historical details. The site is fully responsive to ensure a smooth experience across devices.

## Live Demo
https://pasiphique.github.io/Pittsburgh-History/

## Features

### Intro Page  
Sets the stage for exploring Pittsburgh’s history with a clean, welcoming layout.

### Interactive Neighborhood Map  
Clickable neighborhoods reveal historical insights and fun facts.

### Fun Facts Page  
Engaging historical details presented in a visually appealing format.

### Responsive Design  
Optimized for desktops, tablets, and mobile devices.

### Tech Stack
React, Javascript, HTML, CSS


Follow these steps to run the project locally:

1. Clone the repository
    - git clone link
2. Install dependencies
    - npm install
3. start the development server
    - npm run dev
